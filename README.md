# chromiumbb — Chromium build, but better

A modern replacement for Chromium's `depot_tools` / `gclient` / `gn` / `ninja` toolchain.

**Goals:**
- Simplified single-command builds
- Faster dependency management
- Saner defaults for privacy/security-focused builds
- Drop-in compatible with existing Chromium build workflows

Built for [Acreetium](https://github.com/spivanatalie64/acreetium), the Chromium engine of AcreetionOS.
